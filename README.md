# Federated Learning vs Homomorphic Encryption for Clinical Prediction

This experiment suite compares centralised machine learning, federated learning, and CKKS homomorphic-encryption-based inference for privacy-preserving clinical prediction.

## Datasets

- UCI Cleveland Heart Disease
- Stroke Prediction Dataset

## Methods Compared

1. Centralised Logistic Regression
2. Federated Logistic Regression using FedAvg
3. CKKS Homomorphic-Encryption-Based Inference

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Balanced accuracy
- ROC-AUC
- Runtime
- Communication overhead
- Deployment suitability

## Scope

This study extends the Privacy-Preserving AI Lab experiments by adding simulated multi-hospital federated training, cross-method comparison, communication-cost measurement, and deployment-oriented analysis.

It is designed as a separate comparative study, not as a duplicate of the existing PPA-AI framework experiment.
